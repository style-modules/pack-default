> The Default Pack for Style Modules

## Getting Started
Install via npm
```shell
npm install --save style-modules/pack-default#0.1.0
```

Or include via RawGit
```html
<link rel="stylesheet" href="https://cdn.rawgit.com/style-modules/pack-default/0.1.0/bundle/main.css" />
```
